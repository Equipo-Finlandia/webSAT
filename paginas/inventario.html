1.Tablero Pedidos:

En este departamento puedes organizar y realizar operaciones de stocks

1.1Transferencias

Es Transferencias puedes consultar todas las trasferencias registradas, con atributos referencia, empresa, fecha, origen
y estado


  1. El boton "Crear" nos permite crear una nueva Operacion.
  2. Esta barra nos permite organizar el orden y visualizar las operaciones, puedes hacer click en zona de encabezado corrrespondiente.
  3. Buscador de operaciones, para utilizarlo, introduce la parabla clave de búsqueda y seleccionar un filtro
    correspondiente(Empresa, Estado, Referencia, etc...) para facilitar la búsqueda.
  4. Estos tres botones nos permite cambiar la forma de visualizacion.

1.2Ajuste de Inventario

En la pestaña Ajuste de Inventario puede visualizar los inventarios actuales. El boton "Crear" nos permite crear un nuevo inventario

  1. Una vez en crear rellenamos el formulario
  2. Seleccionamos el boton Guardar

1.3Desechar:

En la pestaña "Desechar" tenemos la opcion de desechar productos de el inventario.Desechar un nuevo producto seleccionaos "Crear"

  1. Una vez en crear rellenamos el formulario
  2. Seleccionamos el boton Guardar

1.4Ejecutar Planificador

Al sellecionar la pestaña Ejecutar Planificador se abrira una nueva ventana

Simplemente tendremos que selecionar la opcion Ejecutar Planificadores

Productos

En la pestaña Productos, podra visualziar sus productos, editarlos o crear nuevos:

  1. Para crear un produto nuevo seleccione "Crear"
  1. Se abrira un nuevo formulario que tiene que rellenar
  2. Despues de rellenar el formulario pulse en "Guardar"

Reglas de abastecimiento:

En la pestaña de Reglas de abastecimiento podremos visualizar las reglas anteriores o crear nuevas

Para crear una nueva simplemente seleccione "Crear"

  1. Se abrira un nuevo formulario que tiene que rellenar
  2. Despues de rellenar el formulario pulse en "Guardar"

Inventario

En la pestaña de Inventario se abre automaticamente una pequeña pestaña en la que seleccionaremos Recuperar las cantidades de inventario

  1. Se abrira una lista con todos los productos en el inventario.
  2. Su ubicacion, la cantidad reservada, la cantidad a mano

Valoracion del inventario

En esta pestaña se abrira una nueva ventana con varias opciones

  1. Seleccione Retrive the inventory validation.
  1. Para crear una nueva valoracion del inventario simplemente seleccione "Crear"
  2. Se abrira una lista con todos los productos en el inventario y sus valoraciones.

Movimiento:

En la pestaña Movimiento, podra visualizar los movimientos realizados

  1. Seleccione uno y se le abrira una pequeña descripcion.

Almacen

En esta pestaña le apareceran todos los almacenes ya creados. Seleccione uno para editarlo

  1. Se abrira una nueva ventana con un formulario
  2. Despues de rellenar el formulario sellecione "Guardar"

Tipos de operaciones

En esta seccion podremos ver las operaciones realizadas

  1. Ademas puede crear una nueva operacion
  1. Se abrira una nueva ventana con un formulario que tiene que rellenar.
  2. Despues de rellenar el formulario seleecione "Guardar".

Categorias de productos

En esta seccion puede visualizar las Categorias creadas o crear una nueva

  1. Se abrira una nueva ventana con un formulario que tiene que rellenar.
  2. Despues de rellenar el formulario seleecione "Guardar".
